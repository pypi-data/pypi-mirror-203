# v

Welcome to v - essential tools for programming in Python!

## Installation

Linux / macOS (python3):

```sh
python3 -m pip install v
```

Linux / macOS (python2):

``` sh
python -m pip install v
```

Windows:

``` sh
py -m pip install v
```
